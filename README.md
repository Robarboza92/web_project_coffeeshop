# Triple Espresso

Coffeeshop
Site da cafeteria triple espresso na biblioteca Triple Peaks, nele é possivel observar algumas receitas, reservar uma mesa, verificar o cardaprio, conhecer um pouco sobre o café.
Neste projeto foi utilizado HTML para implementar as informações necessarias, adicionar videos de outro servidor por iframes, criar um formulario através de form para reserva e apresentar o menu do café. E com o CSS foi feita toda a estilização, detalhes, cores, inclusão de imagens de undos e posicionamento dos textos, imagens, icones etc.
